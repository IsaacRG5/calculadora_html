# Calculadora Web
### Autores:
- Issac Ortiz
- Luis Angel Fuentes

## Descripción
Este proyecto consiste en una calculadora web básica desarrollada con **HTML**, **CSS** y **JavaScript**.  
Permite realizar operaciones matemáticas simples como:

- Suma
- Resta
- Multiplicación
- División

La interfaz cuenta con cuatro campos principales:

1. Primer número
2. Operación
3. Segundo número
4. Resultado

Además, incluye botones numéricos y operadores para facilitar el ingreso de datos, así como navegación entre campos mediante la tecla **Enter**.

---

## Tecnologías utilizadas
- **HTML5** → estructura de la calculadora
- **CSS3** → estilos y diseño visual
- **JavaScript** → lógica de operaciones y funcionalidad interactiva

---

## Funcionalidades
- Ingreso de números mediante botones
- Selección manual de la operación matemática
- Resultado automático al presionar "="
- Botón para limpiar todos los campos
- Navegación entre inputs usando la tecla **Enter**
- Diseño centrado y responsive básico
